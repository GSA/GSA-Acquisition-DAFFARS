5301.404 Class Deviations

(b)(i) USD(A&S)/DPCAP is the approval authority for any class deviation described in DFARS 201.402(1), class deviations from DFARS 215.3, and class deviations that meet the criteria in DFARS 201.404(b)(ii)(A-D).

(ii) The cognizant HCA is the approval authority for class deviations from FAR 15.3, DAFFARS 5315.3, and MP5315.3. Requests must be submitted through the SCO to the cognizant HCA workflow for approval.

(iii) SCOs are authorized to approve class deviations except as restricted by paragraphs (i) and (ii) above or DPCAP directed data system requirements. Class deviation approval authority is not further delegable.
